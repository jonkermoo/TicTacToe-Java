# Tic Tac Toe (Java Swing)

This is a 2-player desktop Tic Tac Toe game built with Java Swing.

## Open a Terminal

Navigate to the root project folder

## How to Run

1. Compile: javac -d bin src/*.java

2. Run: java -cp bin App

## Features
- X vs O game with GUI
- Win/tie detection
- Restart button
